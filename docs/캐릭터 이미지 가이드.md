# 캐릭터 이미지 에셋 가이드

캐릭터 소개 페이지에서 사용되는 이미지 파일들의 스펙과 배치 가이드입니다.

## 필요한 이미지 파일

모든 이미지는 `/public` 폴더에 배치해야 합니다.

### 1. Luna 이미지

#### `luna-idle.png` (정적 이미지)
- **용도**: 기본 대기 상태의 Luna 캐릭터
- **설명**: 루나가 자신의 방에서 편안하게 서있는 모습
- **권장 크기**: 300-400px (width) x 400-500px (height)
- **포맷**: PNG (투명 배경 권장)
- **스타일**: 전신 또는 상반신 캐릭터 일러스트

#### `luna-talking-sprite.png` (스프라이트 시트) ✅
- **용도**: AI 인사말 생성/표시 중 말하는 애니메이션
- **구조**: 4x4 그리드 (총 16개 프레임)
- **프레임 구성**:
  - 왼쪽에서 오른쪽으로, 위에서 아래로 순차 재생
  - 프레임 1-4 (첫째 줄): 입 닫힘 → 벌림 → 활짝 → 닫힘
  - 프레임 5-8 (둘째 줄): 말하기 동작 반복
  - 프레임 9-12 (셋째 줄): 표정 변화 또는 추가 동작
  - 프레임 13-16 (넷째 줄): 다시 기본 자세로 복귀
- **크기**: 유동적 (각 프레임 비율만 유지)
  - 예: 전체 1200px x 1600px (각 프레임 300px x 400px)
- **포맷**: PNG (투명 배경 권장)
- **애니메이션**: 자동으로 순환 재생 (0.08초/프레임, 총 1.28초)

4. 각 프레임을 그리드에 맞춰 정확히 배치

## 추가 커스터마이징

### 프레임 수 변경
`CharacterSprite.jsx` 파일에서 다음 값을 수정:
```javascript
const spriteColumns = 4  // 가로 프레임 수
const spriteRows = 4     // 세로 프레임 수
// 총 프레임 수 = spriteColumns × spriteRows
```

예: 3x5 그리드 (15프레임)로 변경하려면:
```javascript
const spriteColumns = 3
const spriteRows = 5
```

### 애니메이션 속도 조절
`CharacterSprite.jsx` 파일에서 다음 값을 수정:
```javascript
const frameDuration = 0.08  // 초 단위 (0.08 = 80ms)
// 총 애니메이션 시간 = frameDuration × 프레임 수
```

## 참고사항

- 스프라이트 시트는 **4x4 그리드** (가로 4개, 세로 4줄) 형태입니다
- 각 프레임은 **동일한 크기**여야 합니다
- 프레임 순서: **왼쪽→오른쪽, 위→아래** (읽기 순서)
- 투명 배경 사용 시 더 자연스러운 결과물
- 고해상도 디스플레이를 위해 @2x 크기로 제작 후 CSS에서 축소 권장

## 4x4 그리드 스프라이트 예시

```
┌───────┬───────┬───────┬───────┐
│   1   │   2   │   3   │   4   │ ← 첫째 줄
├───────┼───────┼───────┼───────┤
│   5   │   6   │   7   │   8   │ ← 둘째 줄
├───────┼───────┼───────┼───────┤
│   9   │  10   │  11   │  12   │ ← 셋째 줄
├───────┼───────┼───────┼───────┤
│  13   │  14   │  15   │  16   │ ← 넷째 줄
└───────┴───────┴───────┴───────┘
```

애니메이션 재생 순서: 1 → 2 → 3 → 4 → 5 → ... → 16 → 1 (반복)
